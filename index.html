Step 1

Define the System

Step 2

Identify Energy Types

Step 3 — Path A

Conservative Only → Use Ei = Ef

Mechanical energy is conserved. Ei = Ef.

Step 3 — Path B

Non-Conservative → Work-Energy Theorem

Use Ef = Ei + WEXT.

Step 4

Calculate Work

Use WEXT = F · d · cos θ.

Step 5

Solve for Unknown

Plug in values and check units (Joules).

Find speed at bottom (m=2kg, h=10m):

  1. System: ball + Earth.
  2. Ei = Ef
  3. PEg,i + KEi = PEg,f + KEf
    mgh + 0 = 0 + ½mv²
    196 = v² → v = 14 m/s

Find final speed (m=2kg, h=3m, ramp=6m, μ=0.2):

  1. WEXT = −fk·d = −20.4 J
  2. Ef = Ei + WEXT
  3. ½mv² = PEg,i + WEXT
    ½(2)v² = 58.8 − 20.4
    v = √38.4 = 6.2 m/s

Spring Launch (k=200, Δx=0.15):

  1. Ei = PEs,i = ½kΔx²
  2. Ef = PEg,f = mgh
  3. Ei = Ef → h = 0.23m
