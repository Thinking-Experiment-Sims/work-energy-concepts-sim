⚠ Common Misconceptions

❌ "If the object moves, energy was added." — Motion alone doesn't mean external work. See Ramp with μ=0 or Free Fall.
❌ "Gravity always adds energy." — In the ball+Earth system, gravity is internal. It transforms energy, doesn't add it.
✅ External work requires a force with a component parallel to displacement, coming from outside the defined system.

The highlighted step is the key method for the currently selected scenario.

Step 1

Define the System

Choose what's included: Object only? Object + Earth (adds U_g)? Object + Spring (adds U_s)? Your system choice determines which forces are internal vs. external.

Step 2

Identify Energy Types Present

Which energies exist? KE (moving), U_g (height), U_s (spring), E_thermal (friction)?

Step 3 — Path A

Conservative Forces Only → Use E_i = E_f

If only gravity/spring/tension act (no friction, no applied push), total mechanical energy is conserved. E_i = E_f → solve directly.

KE_i + U_g,i + U_s,i = KE_f + U_g,f + U_s,f

Step 3 — Path B

Non-Conservative Forces Present → Use Work-Energy Theorem

If friction or an applied force acts, use E_f = E_i + W_ext where W_ext is the sum of all non-conservative work.

E_f = E_i + W_friction + W_applied

Step 4

Calculate Work (if needed)

Use W = F · d · cos θ where θ is the angle between the force vector and displacement vector.

θ=0° → W=Fd (max)  |  θ=90° → W=0  |  θ=180° → W=−Fd

Step 5

Solve for the Unknown

Plug in values, solve algebraically, check units (Joules), and verify reasonableness.

Example 1 — Free Fall

A 2 kg ball is dropped from 10 m. Find its speed at the bottom.

  1. Define system: ball + Earth. Gravity is internal. No external work.
  2. Energies: KE and U_g only. No spring.
  3. Conservative only → use E_i = E_f.
  4. mgh_i + 0 = 0 + ½mv²
    (2)(9.8)(10) = ½(2)v²
    196 = v² → v = 14 m/s
  5. Check: units are m/s ✓. Mass cancels — speed is independent of mass ✓.

A 2 kg block slides down a 30° ramp (length 6 m) with μ_k = 0.2. Find the final speed.

  1. System: block + Earth. Height h = 6·sin30° = 3 m.
  2. Energies: KE, U_g. Friction is non-conservative.
  3. Use Work-Energy Theorem: E_f = E_i + W_friction.
  4. W_friction = −μ_k·mg·cosα·d = −(0.2)(2)(9.8)(cos30°)(6) = −20.4 J
    E_i = mgh = (2)(9.8)(3) = 58.8 J
    E_f = 58.8 − 20.4 = 38.4 J = ½mv²
    v = √(2·38.4/2) = 6.2 m/s
  5. Compare to frictionless: v = √(2gh) = 7.67 m/s. Friction reduced final speed ✓.

A spring (k=200 N/m) is compressed 0.15 m and launches a 1 kg block upward. Find max height.

  1. System: block + Earth + spring. Conservative only.
  2. Energies: U_s initially, U_g finally (KE = 0 at max height).
  3. E_i = E_f → U_s,i = U_g,f.
  4. ½kΔx² = mgh
    ½(200)(0.15)² = (1)(9.8)h
    2.25 = 9.8h → h = 0.23 m
  5. Check: If k doubles, h doubles. If Δx doubles, h quadruples (quadratic) ✓.

A pendulum (L=1.5 m) is released from 40°. Find v at the bottom.

  1. System: bob + Earth. Tension ⊥ motion → W_T = 0. Conservative.
  2. h = L(1−cosθ) = 1.5(1−cos40°) = 1.5(1−0.766) = 0.351 m.
  3. E_i = E_f → mgh = ½mv² → v = √(2gh).
  4. v = √(2·9.8·0.351) = √6.88 = 2.62 m/s
  5. Mass cancels — speed at bottom is independent of bob mass ✓.

A 20 N force pushes a block 4 m at θ=30° above horizontal. Find work done by the force.

  1. Identify: F=20 N, d=4 m, θ=30° (between force and displacement).
  2. Only the horizontal component F·cosθ does work along the horizontal displacement.
  3. Apply W = F·d·cosθ.
  4. W = (20)(4)cos(30°) = 80 × 0.866 = 69.3 J
  5. At θ=90°: W = 80·cos90° = 0. The force is perpendicular → does zero work ✓.
